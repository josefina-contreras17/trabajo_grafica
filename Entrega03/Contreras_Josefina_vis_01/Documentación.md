Aparte de la visualización y su crónica, deben crear un archivo específico donde consignen la documentación del trabajo que realizaron. Esta documentación es de vital importancia para la calidad de su reportaje, ya que asegura la transparencia y replicabilidad de su proceso periodístico. Estos registros incluyen:

Archivo README de la entrega personal:
1. Explicación de cómo se realizó el proceso de visualización, con los pasos y descinones tomados.
2. Base de datos (CSV) que utilizaron, cómo la procesaron hasta dejarla lista para usar en la visualización que quieren hacer y por qué la seleccionaron.
3. Ejemplos sobre preguntas que puede responder su visualización de datos final (o el conjunto de visualizaciones que crearon).
4. Ficha técnica de la base de datos:
    1. Característica de los datos si es necesario
    2. Variables incorporadas: variable | descripción
    3. Observaciones que tengan sobre la base de datos.
    4. Base de datos en formato CSV
